# GWTC's Visualizer v30.
Audio visualizer that is producing nearly identical visual result of the After Effects Audio Spectrum effect, but in real time.
Signature music visualizer of my YouTube channel GWTC, that is now available for everyone for free.

<img width="1826" height="877" alt="image" src="https://github.com/user-attachments/assets/3de39051-68fb-420e-a69b-4da2b8f8ea84" />

This is the public beta, and the app is in development. It supports various customization settings, as well as additional effects, such as glow filters, and fullscreen flashes. Comes with presets, for an easier understanding of the custom preset creation. 

I do plan make some updates in time, and currently this build has some various bugs, but still very solid performance. If you found any bugs that you want to report - I will appreciate this, and you can leave the feadback here or to my email - goldenwolfchannel@gmail.com.
